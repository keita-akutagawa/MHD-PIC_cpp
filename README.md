# MHD-PIC

連結階層シミュレーションのコードです。 \
IdealMHDとPICを接続領域を用いて繋げます。 \
1次元は、MHD→PIC・PIC→MHDのfast modeの伝搬に成功しました。 \

C++で書かれています。 \
